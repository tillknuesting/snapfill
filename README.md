# PDF Helper

A browser-side tool for filling, signing, drawing on, and editing PDFs.
Everything runs locally — files never leave the device.

## What it does

- **Fill** native form fields, drop free text anywhere, snap text to detected
  rows and input boxes on a form.
- **Sign** by drawing with mouse or touch, or by typing your name in one of
  several handwriting fonts. Save signatures for reuse.
- **Draw** freehand with adjustable colour, width, and opacity.
- **Stamp dates** in a chosen locale (system / US / UK / DE / ISO).
- **Auto-fill profile** values (name, address, IBAN, …) into the next click.
- **Edit existing PDF text** — click a word or paragraph, type the
  replacement; bold/italic, colour, paragraph alignment, and multi-line
  layout are preserved into the downloaded file.
- **Insert images** for the current session.
- **Multi-page** PDFs with thumbnail navigation, zoom, full undo / redo, and
  per-file persistence so edits survive a reload.
